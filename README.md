# LaunchFoundry Lite

A local-first campaign dashboard that turns a website/project folder concept into marketing strategy, storyboards, Remotion-ready render specs, publishing copy, and QA notes.

## First test case

**18th Century Woodworking Services** → Facebook Reel campaign.

## Run locally

```bash
npm install
npm run dev
```

## Included

- React + Vite + TypeScript
- localStorage persistence
- mock 18th Century Woodworking project
- Campaign Prompt page
- Project Scan page
- Brand Profile page
- Audience Strategy page
- Campaign Concepts page
- Storyboard page
- Render Spec JSON page
- Publishing Pack page
- QA page
- Claude/Cowork prompt templates
